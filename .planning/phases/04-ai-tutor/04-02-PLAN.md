    Create `frontend/components/TutorChat.tsx`. This is a "use client" component.

    **Props:**
    ```tsx
    interface TutorChatProps {
      weekMap: WeekMap  // from dashboard — used to populate week filter options
    }
    ```

    **Local types:**
    ```tsx
    interface WeekMap {
      course_name: string
      weeks: Array<{ week: number; topic: string; readings: string[]; notes?: string }>
    }
    interface ChatMessage {
      role: "user" | "assistant"
      content: string
      citations?: Array<{ filename: string; week_number: number | null; url: string | null }>
    }
    ```

    **State:**
    - `messages: ChatMessage[]` — session-only, starts empty
    - `input: string` — current text field value
    - `loading: boolean` — true while awaiting API response
    - `selectedWeek: number | null` — null = all weeks

    **Week filter dropdown:**
    - First option: "All weeks" (value: "") maps to selectedWeek = null
    - Remaining options: weekMap.weeks mapped as `Week ${w.week}: ${w.topic}` (value: `${w.week}`)
    - On change: parse to int or null and set selectedWeek

    **Send handler:**
    1. If input.trim() is empty or loading is true, return early
    2. Append user message to messages
    3. Set loading = true, clear input
    4. Call `apiFetch("/api/v1/tutor/chat", { method: "POST", body: JSON.stringify({ question: input.trim(), history: messages.slice(-10).map(m => ({ role: m.role, content: m.content })), week_number: selectedWeek }) })`
    5. On success: parse JSON, append assistant message with answer and citations
    6. On error: append assistant message with content "Sorry, something went wrong. Please try again." and empty citations
    7. Set loading = false

    **Message rendering:**
    - User messages: right-aligned bubble, light background (e.g. `bg-blue-600 text-white self-end`)
    - Assistant messages: left-aligned, dark background (e.g. `bg-gray-800 text-gray-100 self-start`)
    - Each message in a flex column; message bubbles use `rounded-lg px-4 py-3 max-w-[85%]`

    **Typing indicator** (shown when loading = true, after any existing messages):
    - An assistant-styled bubble containing three animated dots
    - Use CSS animation with `animate-bounce` staggered delays on three spans:
      ```tsx
      <span className="inline-block w-2 h-2 bg-gray-400 rounded-full animate-bounce [animation-delay:0ms]" />
      <span className="inline-block w-2 h-2 bg-gray-400 rounded-full animate-bounce [animation-delay:150ms]" />
      <span className="inline-block w-2 h-2 bg-gray-400 rounded-full animate-bounce [animation-delay:300ms]" />
      ```

    **Citations block** (rendered below assistant message prose, only if citations.length > 0):
    ```tsx
    <div className="mt-3 pt-3 border-t border-gray-700 text-sm">
      <p className="text-gray-400 font-medium mb-1">Sources:</p>
      <ul className="space-y-1">
        {message.citations.map((c, i) => {
          const weekEntry = weekMap.weeks.find(w => w.week === c.week_number)
          const label = weekEntry ? `Week ${c.week_number}: ${weekEntry.topic}` : c.week_number ? `Week ${c.week_number}` : ""
          const display = label ? `${c.filename} — ${label}` : c.filename
          return (
            <li key={i}>
              {c.url ? (
                <a href={c.url} target="_blank" rel="noopener noreferrer"
                   className="text-blue-400 hover:text-blue-300 underline transition-colors">
                  {display}
                </a>
              ) : (
                <span className="text-gray-400">{display}</span>
              )}
            </li>
          )
        })}
      </ul>
    </div>
    ```

    **Input area:**
    - `<textarea>` (or `<input type="text">`) for question input; `onKeyDown` sends on Enter (not Shift+Enter)
    - Send button: disabled when `loading || input.trim() === ""`; shows "Send" text
    - Use `ref` or `useEffect` to scroll message list to bottom after each new message

    ```tsx
      {/* Input area */}
      <div className="flex gap-2 pt-4 border-t border-gray-800">
        <input ... />
        <button ...>Send</button>
      </div>
    </div>
    ```
